도움말 창 사용하기

세부 항목:
  • 원하는 주제 찾기
  • 도움말 검색 방법

원하는 주제 찾기

이 도움말 창에서 정보를 보려면, 좌측 목차 탐색 창에서 원하는 주제를 클릭합니다. 각 주제 옆에 있는 플러스 아이콘을 클릭하면 triangle 해당 주제에 대한 세부 항목을 볼 수 있습니다.

도움말 주제를 검색하려면, 단어나 구문을 상단 검색 도구에 입력하여 관련된 주제를 찾아 봅니다. 각 주제를 클릭하여 해당 주제에 대한 내용을 살펴봅니다. 검색 결과가 없으면, 글자 수를 줄여 보거나 단어의 조합을 바꿔 봅니다. 검색 도구에 글자를 입력하면 바로 검색 결과가 좌측 탐색 창에 나타날 것입니다. 처음 도움말 목차를 다시 보려면, 검색 결과가 있는 좌측 탐색 창 상단의 닫기 버튼을 클릭하여 검색 도구을 초기화 하거나, 키보드에서 Esc를 누릅니다.

지금 읽고 있는 도움말 주제와 연관된 정보를 보려면, 보통 웹 페이지와 마찬가지로 링크를 클릭합니다. 또한 뒤로 앞으로 버튼을 이용하여 이미 보았던 페이지로 이동할 수도 있습니다.

도움말 검색 방법

원하는 결과를 찾을 수 없는 경우에는, 다음 검색 방법를 참조합니다:

  • 지금 찾고 있는 검색어가 &brandShortName;와 관련된 것이 맞는지 확인합니다; 여기서 지금 하고있는 검색은 일반적으로 하는 인터넷 검색이 아닙니다.
  • 검색 범위를 넓혀 검색어가 지나치게 구체적이지 않게 합니다; 검색어가 다른 용어로 표현되어 있을 수도 있습니다.
  • 검색 범위를 지나치게 넓히지 않습니다. 예를 들어, "페이지"와 같이 흔하게 사용되는 용어로 검색하면 필요 이상으로 너무 많은 주제가 검색될 가능성이 높습니다.
